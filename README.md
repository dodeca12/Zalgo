### How does zalgo-text work?

Zalgo-text is generated by leveraging the [Unicode](https://en.wikipedia.org/wiki/Unicode) [diacritical marks](https://en.wikipedia.org/wiki/Diacritic) glyph concatenation hack to generate zalgo-text such alike the ever infamous H̢͕̞̠͐̎ͨ͠͡e̶ͪ͐̌͏̢͔̘̰ c̛̓ͨ̇͏̶̗̤ͅo̡̬̘̗̎̑ͮ̀͜ṃ̶̨̪͎̐ͪͧ͞ȩ̧̹͖̹̽͛̾͟s̶̷̟̭̪̿̆͂͞  - a common stay among internet image boards.  

Diacritical glyphs concatenation in Unicode works by concatenating a particular glyph of a [grapheme](https://en.wikipedia.org/wiki/Grapheme) (for this example, we use the [Latin alphabet](https://en.wikipedia.org/wiki/Latin_alphabet)) with the glyph of a diacritical mark to create a grapheme with a diacritical mark.
